
# BK4 Digital Twin – Predictive Maintenance

## Overview

This repository contains the source codes developed as part of my Master's Thesis in Smart Industry.

The project focuses on the development of a virtual Digital Twin of the BK4 Vertical Roller Mill at Holcim Fès for predictive maintenance and intelligent supervision.

## Technologies Used

- MATLAB
- Unity 3D
- Blender
- C#
- UDP Communication
- Machine Learning (Support Vector Machine - SVM)

## Repository Contents

- Unity C# scripts
- MATLAB scripts
- MATLAB Function codes
- Project documentation
- Images and screenshots

## Project Features

- 3D visualization of the BK4 vertical roller mill
- Bidirectional communication between MATLAB and Unity
- Predictive maintenance algorithms
- SVM-based fault diagnosis
- Real-time monitoring dashboard
